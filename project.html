Securing Virtual Market Places

The comprehensive fraud detection system incorporates a blend of machine learning, deep learning, and genetic algorithms to effectively identify and classify fraudulent activities in both online transactions and website authenticity. By analyzing various features such as URL age, presence of anchor tags, and transaction patterns, this advanced system aids users in determining the legitimacy of websites and prevents payment fraud in credit card transactions.


Utilizing a diverse range of algorithms including Decision Trees, Random Forest, Deep Neural Networks (DNN), Multilayer Perceptron (MLP), and genetic algorithms, the system ensures robust detection and classification capabilities. This adaptable solution can be seamlessly integrated into web applications, offering broad utility for users seeking enhanced security against both phishing attempts and fraudulent transactions.

Forecasting COVID 19 using ARIMA Model

we provide technique for predicting upcoming cases based on historical information. Two strategies, one for predicting the likelihood of contracting the infection and the other for forecasting the number of positive cases, are described using machine learning techniques. Different algorithms were tested, and we cover the algorithm that produced the most accurate results. It is discussed how to forecast confirmed cases for different Indian states using autoregressive integrated moving average time series(ARIMA).
